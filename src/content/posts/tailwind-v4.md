## Tailwind CSS v4

最新の Tailwind CSS では、設定ファイル（`tailwind.config.js`）が不要になり、CSS 変数をベースにした直感的なカスタマイズが可能になりました。

### 主な変更点

- **Zero-config**: デフォルトで多くの機能が有効。
- **高性能**: ビルド速度が大幅に向上。
- **モダン**: CSS ネイティブな機能を最大限に活用。

ダークモードの対応もより簡単になっています。
